## 🚀 Getting Started

### View the Page

You can view the live menu page here:
👉 **[Visit Cafe Aroma Menu](https://deepakshi336-max.github.io/Restaurnt-menu.page/)**

### Local Setup

1. Clone the repository:
```bash
git clone https://github.com/deepakshi336-max/Restaurnt-menu.page.git
```

2. Navigate to the folder:
```bash
cd Restaurnt-menu.page
```

3. Open `index.html` in your web browser:
```bash
open index.html
```

## ✨ Interactive Elements

### Mobile Menu Toggle
- Click the hamburger menu (≡) to view navigation options
- Smooth transitions and fixed positioning for easy access

### Favorites System
- Click the heart (🤍) on any menu item to add it to favorites
- Heart turns red (❤️) when selected

### Welcome Toast
- Automatic welcome notification on page load
- Disappears after 3 seconds
